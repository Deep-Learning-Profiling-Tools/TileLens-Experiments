# End-to-End Overhead Experiments

This directory contains scripts to measure the overhead of `triton-profiler` and `triton-sanitizer` compared to running Triton kernels with plain Python.

## Scripts

### 1. e2e_runner.py

Runs TritonBench test cases in three modes:

- **baseline**: Run with plain Python
- **profiler**: Run with `triton-profiler`
- **sanitizer**: Run with `triton-sanitizer`

#### Usage

```bash
# Run all modes (baseline, profiler, sanitizer)
python e2e_runner.py --mode all

# Run only baseline
python e2e_runner.py --mode baseline

# Run only profiler
python e2e_runner.py --mode profiler

# Run only sanitizer
python e2e_runner.py --mode sanitizer

# Run multiple modes
python e2e_runner.py --mode profiler sanitizer

# Run a single test case
python e2e_runner.py --mode all --case matmul_triton1

# Run liger_kernel tests instead of tritonbench
python e2e_runner.py --repo liger_kernel --mode all

# Use a custom test registry
python e2e_runner.py --mode all --registry /path/to/registry.txt
```

#### Options

| Option | Description |
|--------|-------------|
| `--repo` | Repository to run tests from: `tritonbench` or `liger_kernel` (default: `tritonbench`) |
| `--mode`, `-m` | Mode(s) to run: `baseline`, `profiler`, `sanitizer`, or `all` (default: `all`) |
| `--registry`, `-w` | Path to test registry file (default: `../utils/test_registry.txt`) |
| `--case`, `-c` | Run a single test case (e.g., `matmul_triton1`) |
| `--output-dir`, `-o` | Base output directory for results (default: current directory) |

#### Output

Results are saved to:
- `baseline/` - Log files for plain Python runs
- `profiler/` - Log files for triton-profiler runs
- `sanitizer/` - Log files for triton-sanitizer runs

A JSON summary file `e2e_summary_<timestamp>.json` is also generated.

---

### 2. collect_e2e_times.py

Collects elapsed times from the log files generated by `e2e_runner.py` and outputs a CSV file.

#### Usage

```bash
# Collect times and generate e2e_times.csv
python collect_e2e_times.py

# Specify output file
python collect_e2e_times.py -o results.csv

# Specify input directory
python collect_e2e_times.py -i /path/to/results -o results.csv
```

#### Options

| Option | Description |
|--------|-------------|
| `--input-dir`, `-i` | Directory containing `baseline/`, `profiler/`, `sanitizer/` folders (default: current directory) |
| `--output`, `-o` | Output CSV file path (default: `e2e_times.csv`) |

#### Output Format

The generated CSV file has the following columns:

| case | baseline (s) | profiler (s) | sanitizer (s) |
|------|--------------|--------------|---------------|
| matmul_triton1 | 1.23 | 45.67 | 12.34 |
| softmax_triton1 | 0.89 | 23.45 | 8.90 |

---

## Workflow

1. **Run experiments**:
   ```bash
   python e2e_runner.py --mode all
   ```

2. **Collect results**:
   ```bash
   python collect_e2e_times.py
   ```

3. **View results**:
   ```bash
   cat e2e_times.csv
   ```
